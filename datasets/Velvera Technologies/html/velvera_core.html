Common Features

  • Operating System: VelveraOS Lite (Android 13-based, optimized for performance and battery)
  • Build: Reinforced polycarbonate unibody, Gorilla Glass 5 front
  • SIM: Dual Nano SIM + microSD (dedicated)
  • Network: 4G LTE (Core M Series), 5G (Core Max/Plus Series)
  • Security: Side-mounted fingerprint sensor, Face Unlock
  • Audio: Stereo speakers, 3.5mm headphone jack, noise-canceling microphones
  • Connectivity: Wi-Fi 802.11ac, Bluetooth 5.1, GPS/GLONASS
  • Ports: USB-C 2.0, OTG supported
  • Other: Water-repellent nano-coating (IP52), AI system optimization, 2 years security updates
